# part()

Creates an isolation boundary so shapes inside the part stay separate from the rest of the scene.

**Returns**: [**`SceneObject`**](/docs/api/types/scene-object)

:::tip
See the [guide](/docs/guides/3d-operations/parts) for detailed usage examples.
:::

## Signatures

```ts
part(name: string, callback: () => void): SceneObject
```

Creates an isolation boundary so shapes inside the part stay separate from the rest of the scene.

| Parameter | Type | Description |
|-----------|------|-------------|
| `name` | string | The part name. |
| `callback` | () =&gt; void | Callback containing the part geometry. |
