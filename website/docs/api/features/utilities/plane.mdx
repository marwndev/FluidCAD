# plane()

Creates a plane with transform options.

**Returns**: [**`Plane`**](/docs/api/types/plane)

## Signatures

```ts
plane(plane: PlaneLike, options: PlaneTransformOptions): Plane
```

Creates a plane with transform options.

| Parameter | Type | Description |
|-----------|------|-------------|
| `plane` | [PlaneLike](/docs/api/types/plane-like) | The standard plane or normal vector |
| `options` | [PlaneTransformOptions](/docs/api/types/plane-transform-options) | Transform options (offset, rotate, etc.) |

#### PlaneTransformOptions

| Property | Type | Description |
|----------|------|-------------|
| `offset` | number | Distance to translate the plane along its normal *(optional)* |
| `rotateX` | number | Rotation around the plane's X axis (in degrees) *(optional)* |
| `rotateY` | number | Rotation around the plane's Y axis (in degrees) *(optional)* |
| `rotateZ` | number | Rotation around the plane's Z axis / normal (in degrees) *(optional)* |

```ts

```

Creates a plane midway between two standard planes or normal vectors.

| Parameter | Type | Description |
|-----------|------|-------------|
| `p1` | [PlaneLike](/docs/api/types/plane-like) | The first standard plane or normal vector |
| `p2` | [PlaneLike](/docs/api/types/plane-like) | The second standard plane or normal vector |
| `options` | [PlaneTransformOptions](/docs/api/types/plane-transform-options) | Transform options (offset, rotate, etc.) *(optional)* |

```ts

```

Creates a plane midway between two existing Plane objects.

| Parameter | Type | Description |
|-----------|------|-------------|
| `p1` | [Plane](/docs/api/types/plane) | The first Plane object |
| `p2` | [Plane](/docs/api/types/plane) | The second Plane object |
| `options` | [PlaneTransformOptions](/docs/api/types/plane-transform-options) | Transform options (offset, flip, sticky, etc.) *(optional)* |


:::note
All signatures also accept an optional `targetPlane` parameter as the last argument to draw on a specific plane.
:::
