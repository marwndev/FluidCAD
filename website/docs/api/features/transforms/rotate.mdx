# rotate()

[2D] Rotates geometry by an angle inside a sketch.

**Returns**: [**`SceneObject`**](/docs/api/types/scene-object)

:::tip
See the [guide](/docs/guides/transforms) for detailed usage examples.
:::

## Signatures

```ts
rotate(angle: number, ...targets: SceneObject[]): IRotate
```

[2D] Rotates geometry by an angle inside a sketch.

| Parameter | Type | Description |
|-----------|------|-------------|
| `angle` | number | The rotation angle in degrees |
| `...targets` | [SceneObject](/docs/api/types/scene-object)[] | The geometries to rotate (defaults to last object) *(optional)* |

---

```ts
rotate(angle: number, copy: boolean, ...targets: SceneObject[]): IRotate
```

[2D] Rotates geometry by an angle inside a sketch, optionally making a copy.

| Parameter | Type | Description |
|-----------|------|-------------|
| `angle` | number | The rotation angle in degrees |
| `copy` | boolean | Whether to copy instead of move |
| `...targets` | [SceneObject](/docs/api/types/scene-object)[] | The geometries to rotate (defaults to last object) *(optional)* |

---

```ts
rotate(axis: AxisLike, angle: number, ...targets: SceneObject[]): IRotate
```

[3D] Rotates objects around an axis by an angle.

| Parameter | Type | Description |
|-----------|------|-------------|
| `axis` | [AxisLike](/docs/api/types/axis-like) | The axis to rotate around |
| `angle` | number | The rotation angle in degrees |
| `...targets` | [SceneObject](/docs/api/types/scene-object)[] | The objects to rotate (defaults to last object) *(optional)* |

---

```ts
rotate(axis: AxisLike, angle: number, copy: boolean, ...targets: SceneObject[]): IRotate
```

[3D] Rotates objects around an axis by an angle, optionally making a copy.

| Parameter | Type | Description |
|-----------|------|-------------|
| `axis` | [AxisLike](/docs/api/types/axis-like) | The axis to rotate around |
| `angle` | number | The rotation angle in degrees |
| `copy` | boolean | Whether to copy instead of move |
| `...targets` | [SceneObject](/docs/api/types/scene-object)[] | The objects to rotate (defaults to last object) *(optional)* |
