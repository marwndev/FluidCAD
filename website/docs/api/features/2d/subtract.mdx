# subtract()

Subtracts the second shape from the first (boolean difference).
Works with both 3D solids and 2D sketch geometries.

**Returns**: [**`SceneObject`**](/docs/api/types/scene-object)

:::tip
See the [guide](/docs/guides/booleans-and-fusion) for detailed usage examples.
:::

## Signatures

```ts
subtract(object1: SceneObject, object2: SceneObject): SceneObject
```

Subtracts the second shape from the first (boolean difference).
Works with both 3D solids and 2D sketch geometries.

| Parameter | Type | Description |
|-----------|------|-------------|
| `object1` | [SceneObject](/docs/api/types/scene-object) | The base shape |
| `object2` | [SceneObject](/docs/api/types/scene-object) | The shape to subtract |
