# intersect()

Intersects 3D objects with the current sketch plane, producing cross-section edges.

**Returns**: [**`ExtrudableGeometry`**](/docs/api/types/extrudable-geometry)

## Signatures

```ts
intersect(...sourceObjects: SceneObject[]): ExtrudableGeometry
```

Intersects 3D objects with the current sketch plane, producing cross-section edges.

| Parameter | Type | Description |
|-----------|------|-------------|
| `...sourceObjects` | [SceneObject](/docs/api/types/scene-object)[] | The 3D objects to intersect *(optional)* |

---

```ts
intersect(targetPlane: SceneObject | PlaneLike, sourceObjects: SceneObject[]): ExtrudableGeometry
```

Intersects 3D objects with a target plane, producing cross-section edges.

| Parameter | Type | Description |
|-----------|------|-------------|
| `targetPlane` | [SceneObject](/docs/api/types/scene-object) \| [PlaneLike](/docs/api/types/plane-like) | The plane to intersect with |
| `sourceObjects` | [SceneObject](/docs/api/types/scene-object)[] | The 3D objects to intersect |
