# project()

Projects 3D objects onto the current sketch plane.

**Returns**: [**`ExtrudableGeometry`**](/docs/api/types/extrudable-geometry)

:::tip
See the [guide](/docs/guides/sketching/projection) for detailed usage examples.
:::

## Signatures

```ts
project(...sourceObjects: SceneObject[]): ExtrudableGeometry
```

Projects 3D objects onto the current sketch plane.

| Parameter | Type | Description |
|-----------|------|-------------|
| `...sourceObjects` | [SceneObject](/docs/api/types/scene-object)[] | The 3D objects to project *(optional)* |

---

```ts
project(targetPlane: SceneObject | PlaneLike, sourceObjects: SceneObject[]): ExtrudableGeometry
```

Projects 3D objects onto a target plane.

| Parameter | Type | Description |
|-----------|------|-------------|
| `targetPlane` | [SceneObject](/docs/api/types/scene-object) \| [PlaneLike](/docs/api/types/plane-like) | The plane to project onto |
| `sourceObjects` | [SceneObject](/docs/api/types/scene-object)[] | The 3D objects to project |
