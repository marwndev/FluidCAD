# circle()

Draws a circle at a given center with an optional diameter.

**Returns**: [**`ExtrudableGeometry`**](/docs/api/types/extrudable-geometry)

:::tip
See the [guide](/docs/guides/sketching/primitive-shapes) for detailed usage examples.
:::

## Signatures

```ts
circle(center: Point2DLike, diameter?: number): ExtrudableGeometry
```

Draws a circle at a given center with an optional diameter.

| Parameter | Type | Description |
|-----------|------|-------------|
| `center` | [Point2DLike](/docs/api/types/point2dlike) | The center point |
| `diameter` | number | The circle diameter (defaults to 40) *(optional)* |

---

```ts
circle(diameter?: number): ExtrudableGeometry
```

Draws a circle at the origin with an optional diameter.

| Parameter | Type | Description |
|-----------|------|-------------|
| `diameter` | number | The circle diameter (defaults to 40) *(optional)* |

---

```ts
circle(targetPlane: SceneObject | PlaneLike, diameter: number): ExtrudableGeometry
```

Draws a circle with a given diameter on a specific plane.

| Parameter | Type | Description |
|-----------|------|-------------|
| `targetPlane` | [SceneObject](/docs/api/types/scene-object) \| [PlaneLike](/docs/api/types/plane-like) | The plane to draw on |
| `diameter` | number | The circle diameter |
