# FaceFilter

Creates a new face filter builder for selecting faces by geometric properties.

```ts
import { face } from 'fluidcad/filters';
```

## Methods

### .atIndex()

```ts
.atIndex(index: number): this
```

Selects the face at the given index.

| Parameter | Type | Description |
|-----------|------|-------------|
| `index` | number | Zero-based face index. |

### .notAtIndex()

```ts
.notAtIndex(index: number): this
```

Excludes the face at the given index.

| Parameter | Type | Description |
|-----------|------|-------------|
| `index` | number | Zero-based face index to exclude. |

---

### .onPlane()

```ts
.onPlane(plane: PlaneLike, offset?: number): this
```

Selects faces that lie on the given plane.

| Parameter | Type | Description |
|-----------|------|-------------|
| `plane` | [PlaneLike](/docs/api/types/plane-like) | The reference plane. |
| `offset` | number | Optional distance to offset the plane before matching. *(optional)* |
